OPTIMIZATION PIPELINE

From raw kernel to peak performance

A fully automated multi-round loop: profile, plan, code, compile, benchmark — repeat until optimal.

01 PROFILING
Nsight Profile
Runs nvcc compilation and Nsight Systems profiling to establish baseline execution time and identify memory bottlenecks.
ncu · nsys
02 CLASSIFICATION
Kernel Classifier
Categorizes the kernel type — reduction, stencil, GEMM, elementwise — and retrieves matching optimizations from the knowledge base.
RAG · vector search
03 PLANNING
Strategy Planner
Generates 5 ranked optimization strategies — shared memory tiling, warp shuffle, vectorized loads — ordered by expected gain.
LLM planning
04 CODING
CUDA Coder
Rewrites the kernel in CUDA C++ applying the selected strategy, injecting SM-specific tuning parameters for the target architecture.
CUDA C++ · LLM
05 VALIDATION
Tester & Reflector
Compiles and runs correctness tests. On failure, a reflector agent analyzes the error and auto-patches before retry.
nvcc · correctness check
06 KNOWLEDGE
KnowledgeBase
Stores every successful optimization with its speedup, strategy, and kernel fingerprint — enabling accelerated future runs.
vector DB · retrieval
